# Brightex - The Art of Automotive Shine

## Project info

High-performance polishing pads and surface finishing accessories.

## Development

This project requires Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Install dependencies
npm i

# Start the development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
